# owenshop
This project was developed using the following technologies:
<ul>
  <li>Reactjs</li>
  <li>Tailwindcss</li>
  <li>Php</li>
  <li>MySQl</li>
</ul>

Clone the project and access the folde :
```
 git init 
 git clone https://github.com/riadZk/owenshop.git
 cd owenshop
```
# Start the project
```
`npm start`
```
